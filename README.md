## 📋 项目概述

本项目旨在帮助组员在一个月内，从零基础到能够独立使用 React 框架开发并部署一个美观的个人主页。

- 🎯 **边做边学**：通过实际项目掌握知识

- 📦 **工程化思维**：养成良好的代码规范和文档习惯

## 💡 核心原则

### 关于 AI 使用

**✅ 鼓励的做法：**
- 让 AI 解释代码原理
- 让 AI 帮你优化代码
- 让 AI 解决报错问题
- 让 AI 提供设计建议

**❌ 禁止的做法：**
- 直接复制大段代码而不理解
- 无法向 advisors 解释你的代码
- 提交前不测试 AI 生成的代码

### 关于设计审美

**不要做"工程师式设计"**，而要：
- 参考专业设计作品（Dribbble, Behance, Pinterest）
- 使用成熟的 UI 组件库
- 注重配色、排版、留白
- 多看优秀作品培养审美

### 关于代码规范

- 使用有意义的变量和函数名
- 适当添加注释说明复杂逻辑
- 删除调试用的 `console.log`
- Git commit 信息要具体明确
- 保持代码文件结构清晰
