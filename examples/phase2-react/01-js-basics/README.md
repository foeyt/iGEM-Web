# JavaScript 基础示例

这个目录包含 JavaScript 基础语法和 DOM 操作的示例代码。

## 📁 文件说明

- `basics.js` - JavaScript 基础语法示例
- `dom-manipulation.html` - DOM 操作示例
- `dark-mode-example.html` - 暗黑模式切换完整示例

## 🎯 学习目标

- 掌握 JavaScript 基本语法
- 理解 DOM 操作
- 能够为页面添加交互功能

## 💡 练习建议

1. 在浏览器 Console 中运行 `basics.js` 中的代码
2. 打开 HTML 文件，查看交互效果
3. 修改代码，实验不同的功能
4. 为你的第一阶段简历添加这些交互功能

## 🚀 运行方式

直接用浏览器打开 `.html` 文件，或使用 Live Server。
